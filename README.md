1. Mis on geen?
● Geen on DNA lõik, mis sisaldab infot ühe tunnuse või valgu kohta.
● Geen ütleb rakule: millist valku teha ja millal.
DNA → geen → valk → tunnus (nt silmavärv)
2. DNA, RNA ja valgud
● DNA – pärilikkuse info (asub rakutuumas)
● RNA – DNA info koopia
● Valk – teeb tööd rakus (ensüüm, struktuur, hormoon)
3. Transkriptsioon (DNA → RNA)
Transkriptsioon on protsess, kus:
● DNA info kirjutatakse ümber RNA-ks
● toimub rakutuumas
Näide:
DNA: A T G C
RNA: U A C G
�� RNA-s T asemel on U
4. Translatsioon (RNA → valk)
Translatsioon on protsess, kus:
● RNA info tõlgitakse aminohapeteks
● toimub ribosoomis
RNA loetakse kolmekaupa → need on koodonid
5. Koodon
● Koodon = 3 RNA nukleotiidi
● Iga koodon määrab ühe aminohappe
Näide:
AUG → met (START-koodon6. Aminohapped ja aminoahel
● Aminohapped on valkude ehitusklotsid
● Aminohapped moodustavad aminoahela
● Aminoahel voldib end → valk
Lihtsustus:
DNA ütleb → millised aminohapped
Aminohapped → moodustavad valgu
Valk → määrab tunnuse
7. Genotüüp
● Genotüüp = geenide kombinatsioon
● Tähistatakse tähtedega (nt AA, Aa, aa)
Näide:
● A = pruun silm (dominantne)
● a = sinine silm (retsessiivne)
8. Fenotüüp
● Fenotüüp = see, mis on näha
● sõltub genotüübist
Näide:
● AA → pruun silm
● Aa → pruun silm
● aa → sinine silm
9. Dominantne ja retsessiivne
● Dominantne tunnus avaldub alati (A)
● Retsessiivne avaldub ainult siis, kui on kaks koopiat (aa)
10. Kuidas määrata fenotüüpi genotüübi järgi?
1. Vaata, kas genotüübis on dominantne alleel
2. Kui on → fenotüüp on dominantne
3. Kui pole → retsessiivne
